This repository contains the source code for a Glance widget built with Jetpack Compose. The widget provides information about the device's battery level and, if available, the battery level of connected Bluetooth headphones. It also displays the charging status and allows you to trigger an action to update the widget's data.

Features


Displays the device's battery level as a percentage.
If Bluetooth headphones are connected, it shows their battery level and name.
Indicates the charging status of the device.
Allows you to trigger an action to update the widget's data.
