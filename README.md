# Battery Tracker

The Battery Tracker app is crafted to offer users immediate, live updates regarding their device's available battery information. It leverages the Android-provided battery API to showcase real-time battery percentage details to users.

## Overview

The Battery Tracker app is a utility tool designed for Android devices to provide users with real-time insights into their device's battery status. It offers a simple and intuitive interface to monitor and display essential battery information, including:

- Battery Level: Real-time display of the current battery level.
- Battery Status: Indicates whether the battery is charging, discharging, or at full capacity.
- Temperature and Voltage: Details about the battery's temperature and voltage.
- Broadcast Receiver Integration: The app registers a broadcast receiver to listen for changes in the device's battery status. As the battery level or status changes, the app dynamically updates the displayed information, ensuring real-time accuracy.
- Widget: App also consists of battery widget which will display battery percentage of the phone and any other bluetooth connected device

## TechStack
- Kotlin
- Jetpack Compose
- Battery API
- Glance widget
